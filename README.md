# SPSUL---Web-pro-testy
Webová aplikace pro školní testování na SPŠUL. Každý student dostane unikátní test z náhodně vybraných otázek různých typů a přihlašuje se svým ID. Administrátorské rozhraní umožňuje správu otázek, výsledků i uživatelů s rozdílnou autorizací a možnost přidávání dalších adminů.
